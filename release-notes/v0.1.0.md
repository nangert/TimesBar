# TimesBar v0.1.0

First public release.

## What it does

A native macOS menu bar app for Kimai time tracking. Click the timer icon to:

- See the active timer (project · activity, optional description, live elapsed counter) and stop it with one click.
- Restart one of the 3 most recent stopped entries from Quick start, or open the inline **+ New timer** form to pick any project + activity + note.
- Read today's hours against an 8h target with a progress bar, plus a weekly bar chart with the current day highlighted.

The Kimai API token is stored in the macOS Keychain. The app has no Dock icon (`LSUIElement`) and polls Kimai every 10 seconds; the elapsed counter ticks every second whether or not the dropdown is open.

## Install

### Build from source

```bash
brew install xcodegen
git clone https://github.com/nangert/TimesBar.git
cd TimesBar
make install
open /Applications/TimesBar.app
```

### Direct download

1. Download `TimesBar-v0.1.0.zip` from the assets on this release.
2. Unzip and drag `TimesBar.app` into `/Applications`.
3. First launch is blocked by Gatekeeper. Right-click → **Open** → confirm, or run `xattr -dr com.apple.quarantine /Applications/TimesBar.app`.

## Setup

Click the timer icon, paste a Kimai API token from your Kimai user profile (**Settings → API access**), then **Verify & save**.

## Known limitations

- Kimai base URL is currently hardcoded (`https://times.lipsum.services`). Edit `TimesBar/Networking/KimaiClient.swift` and rebuild if you point at a different Kimai install.
- Ad-hoc signed only — Gatekeeper will warn on first launch (see workaround above).
